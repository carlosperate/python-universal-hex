## [0.1.0] - 2026-01-31

### Added

- Initial release of the Python Universal Hex library.
    - `create_uhex()` function to create Universal Hex files from multiple Intel Hex files.
    - `separate_uhex()` function to split Universal Hex files into individual Intel Hex files.
    - `is_uhex()` function to check if a hex string is a Universal Hex.
- Command-line interface (`uhex`) with `join` and `separate` commands.
- Support for Python 3.9 through 3.14.
